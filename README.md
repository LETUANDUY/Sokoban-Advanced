## Main Features

- **Classic Sokoban Gameplay**
  - Push boxes to target positions
  - Limited move count
  - Star-based level evaluation (1â€“3 stars)

- **Advanced Mechanics**
  - Multiple box types: normal, fire, electric, magnet
  - Interactive environment blocks: doors, conveyors, teleporters, pressure plates, power tiles

- **Energy System**
  - Energy required to play levels
  - Automatic energy regeneration over time
  - Anti-cheat using real-time (NTP-based logic)

- **Daily Missions**
  - 6 daily missions refreshed every day
  - Track player actions and grant rewards
  - Reward milestones based on total points

- **Shop System**
  - Buy energy using in-game coins
  - Purchase and equip character skins

- **Progress Saving**
  - Save player progress, level stars, missions, and settings locally
